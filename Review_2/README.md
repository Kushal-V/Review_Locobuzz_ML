# Review 2 Task

This directory contains the code and resources for Review 2.

## Contents

### 1. IPL Data Analysis
- **Location**: `IPL/`
- **Description**: Analysis of IPL cricket data using Python/Jupyter Notebook.
- **Note**: The raw dataset `IPL.csv` is excluded from the repository due to size constraints.

### 2. Chess Game
- **Location**: `Chess_game/`
- **Description**: A Python implementation of a Chess game, including classes for Board, King, Queen, Rook, Bishop, Knight, and Pawn.
